# Resources — normalising flows

Background reading for the conditioned normalising flow in this directory
(`conversion_flow.py`, `train_flow.ipynb`).

## Papers

- **`1908.09257v4.pdf`** — Kobyzev, Prince & Brubaker, *Normalizing Flows: An Introduction and
  Review of Current Methods* (arXiv:1908.09257, IEEE TPAMI 2021). Surveys the change-of-variables
  construction, coupling and autoregressive transforms, and the spline flows this study builds on.

## Videos

- *Normalizing Flows* — <https://www.youtube.com/watch?v=YPsIq_f_ihQ>. Walks through the core idea
  in depth.
